# GamersPlace
Ruby on Railsでゲーマー同士がコミュニケーション出来るような掲示板アプリを作成してみました。

#機能
・ログイン機能
・お気に入り機能
・レビュー機能
・コメント投稿機能
・検索機能
・ユーザーフォロー機能
